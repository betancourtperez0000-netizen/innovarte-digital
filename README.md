# Innovarte Digital

Landing page premium creada con React, Vite, Tailwind CSS, Framer Motion y Lucide React Icons.

## Instalación

```bash
npm install
npm run dev
```

## Backend PostgreSQL

1. Crea un archivo `.env` a partir de `.env.example`
2. Actualiza `DATABASE_URL` con tu conexión PostgreSQL
3. Ejecuta en una terminal:

```bash
npm run server
```

4. En otra terminal ejecuta la app frontend:

```bash
npm run dev
```

## Estructura principal

- `src/App.jsx`
- `src/main.jsx`
- `src/index.css`
- `src/components/*`
- `index.html`

## Dependencias

- `react`
- `react-dom`
- `framer-motion`
- `lucide-react`
- `tailwindcss`
- `vite`
